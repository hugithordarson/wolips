Release Plan

WOLips development is driven by bug reports and feature requests posted at our bugtracker Jira. Bugs that are not reported won't get fixed, and features that aren't requested will not materialise.

WOLips is a collaborative open source effort. Please consider contributing!

There are three kinds of relases:

  1. Equivalent: 1.2.3 -> 1.2.4 with no new features, only bug fixes.
  2. Compatible: 1.2.3 -> 1.3.0 with new features, and possibly some bugs fixed.
  3. Major: 1.2.3 to 2.0.0 with major additions and/or modifications. Not necessarily compatible with or equivalent to previous versions

Tracker items (bug reports and feature requests) have a Status, a Category and a Group. The Group indicates which release will address the item, and the Category which part of WOProject it concerns. The Status can be one of the following:

  • Open: Just submitted and/or not yet completed.
  • Closed:
  • Deleted:
  • Pending: Waiting for reply from the original author.

To get an idea about when a specific bug will be fixed or a specific feature added look at the tracker item Group:

  • Equivalent: Next Equivalent Release
  • Compatible: Next Compatible Release
  • Major: Next Major Release
  • ?: Not Assigned
