Adding a New Component

 

  • Press the Open The New Wizard button.

or

  • Select the menu option File > New > Other...

 

The New wizard selection dialog appears.

  • Choose WOLips WO 5.1 and 5.2, and then Component.
  • Press the Next button.

 

  • Specify which project you want the new component added to.
  • Name the new component.
  • Press the Finish button.

 

  • Start programming your new WebObjects component.
