Java Coding for the lazy ones

To make java coding even more convenient WOLips adds some templates to the java editor. Let's start with the action template as an example.

Lets create a action named "foo" to return the component "MyComponent".

  1. Enter "act" in a java editor and invoke the code completion with command+space (mac) control+space (other).

  2. Eclipse gives you the option to choose from several completions. Choose "action" and hit return.

  3. Enter "myc" and invoke the code completion.

  4. The java editor expands the "myc" to "MyComponent". In the case there are multiple classes starting with "myc" you'll get a list to choose from. In the case no class starting with "myc" exist you'll get a beep.

  5. Tab to the next box and name your action to "foo".

  6. Another tab and you're on the line to fill to the body.

.

Templates:

  1. action = Create a action to return a page.
  2. action.this = Create a action to return this page.
  3. ec session default = Create a instance variable to hold the editing context from the session.
  4. ec new = Create a new editing context.
  5. new nsarray string = Create a new NSArray inititialised with a String array.
  6. new nsarray int = Create a new NSArray inititialised with a int array.
