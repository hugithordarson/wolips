Building WOLips from source

  1. Install Ant on your system. (Make sure the installation is correct and working properly before continuing!)
  2. Download the latest WOProject source from SourceForge. You may use Eclipse as a CVS client, and should in that case use the woproject/projects directory as your project root. WOLips versions compatible with eclipse 2.x.x are in HEAD and for eclipse 3.x.x you should use the eclipse_3_0_0 branch.
  3. Change to the woproject/projects root directory. (If you want to build from within eclipse you have to set a classpath variable ANT_HOME.)
  4. Create a build.properties file with a property eclipse.home that points to where you installed eclipse.
  5. Execute ant (dist is the default target in build.xml, and that's what you want).
  6. You'll find the result in the ./temp directory.
  7. Install WOLips by pointing the eclipse update manager to ./temp/dist/site.xml
