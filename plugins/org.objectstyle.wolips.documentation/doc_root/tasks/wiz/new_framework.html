Creating a New Framework

 

  • Press the Open The New Wizard button.

or

  • Select the menu option File > New > Other...

 

The New wizard selection dialog appears.

  • Choose WOLips WO 5.1 and 5.2, and then Framework.
  • Press the Next button.

 

  • Name the new framework project, and optionally specify where you want to keep the project files. (The creation options are only relevant if/when importing existing projects.)
  • Press the Finish button.

 

  • Start programming your new WebObjects framework.
